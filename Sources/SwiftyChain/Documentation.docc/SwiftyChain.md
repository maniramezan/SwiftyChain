# ``SwiftyChain``

Store and retrieve keychain values with Swift-native types.

## Overview

SwiftyChain wraps Apple `SecItem*` APIs behind typed keys, a public `Keychain` actor, and a lightweight `@KeychainStorage` property wrapper.

### Core API

- ``Keychain``
